战略咨询 · 看懂顶级咨询公司怎么想问题

方法论、思维框架、报告底稿——别从零造轮子
麦肯锡 McKinsey
战略顶层
"结构化思考 + 事实驱动 + 老板视角"
全球最被引用的战略方法论集散地。核心套路：金字塔原理、MECE 拆解、7S、增长矩阵。报告质量被业内当行业基准抄。
适合：写战略汇报 / 行业研判 / 高管汇报结构 / 学框架套路
Insights 报告库 McKinsey Quarterly
BCG 波士顿咨询
增长 & 矩阵
"BCG 矩阵、增长份额图、消费者洞察"
和麦肯锡同梯队的另一极，视觉化、模型化做得最系统。BCG Matrix（明星/金牛/问题/瘦狗）几乎所有商业 PPT 都用。
适合：品类分析 / 市场份额梳理 / 增长策略 / 业务诊断
Publications BCG 中国
贝恩 Bain & Company
NPS 鼻祖
"客户忠诚度、零售消费品、私募基金视角"
发明了 NPS（净推荐值）。消费品、零售、私募并购领域被引用最多。和凯度合作的中国消费者趋势报告非常实战。
适合：消费者洞察 / 品牌健康度 / 满意度诊断 / 零售研究
Insights 贝恩中国
罗兰贝格 Roland Berger
本土化
"欧洲底色 + 中国市场最深的咨询之一"
汽车、工业、制造业咨询强项。在中国市场深耕多年，报告对中国国情理解深，不像外资"水土不服"。
适合：汽车后市场 / 工业制造 / 出海策略 / 行业扫描
罗兰贝格洞察
埃森哲 Accenture
IT × 战略
"科技咨询 + 战略 + 数字化的综合方案"
全球最大的管理咨询 + 技术服务公司。《技术展望》年度报告被广泛引用，更偏数字化转型和 AI 落地视角。
适合：数字化转型 / AI 落地 / 客户体验 / IT 架构

华与华
符号营销
"超级符号 = 降低传播成本"

适合：品牌命名 / Slogan 创作 / 门店物料 / 全年营销主题
华与华官网 《超级符号》
君智 · 战略定位
竞争战略
"打赢认知战 = 品类代名词"
飞鹤、波司登、雅迪背后的咨询团队。源自特劳特定位理论，但更本土化。强调"在顾客心智中占据一个词"。
适合：竞品对标 / 品类创新 / 品牌定位 / 战略答辩

研报与数据 · 找数据、找案例的源头

一手行业数据来源，写方案引用更有底气
QuestMobile / 易观
互联网数据
"App DAU、用户画像、流量分发数据"
做互联网行业必备数据源。DAU/MAU/留存/竞品对比一应俱全。QuestMobile 偏移动端，易观偏行业分析。
适合：App 行业分析 / 用户行为洞察 / 流量格局
QuestMobile 易观分析
艾瑞 / iResearch
行业研究
"覆盖最广的中文研报平台之一"
互联网、电商、文娱、教育、医疗……几乎每个赛道都有艾瑞报告。引用率最高的本土研报机构之一。
适合：行业白皮书 / 市场规模 / 趋势研判 / PPT 数据引用
艾瑞咨询
晚点 LatePost / 36氪 / 虎嗅
深度报道
"一手商业故事 + 趋势观察"
三家的核心区别：晚点偏独家深度（公司内幕）、36氪偏创投和赛道、虎嗅偏观点和评论。一起订阅能拼出商业全景。
适合：找案例故事 / 行业内幕 / 趋势预判 / 写开篇引子

适合：短视频选题 / 直播带货分析 / 小红书种草 / 投流参考

权威数据
"最硬的官方数据源，写报告必引"

适合：宏观背景 / 政策合规 / 高管汇报"压舱石"

"微软官方 + 优品的本土模板库"

适合：中文汇报 / 工作总结 / 节日庆典 / 部门介绍
