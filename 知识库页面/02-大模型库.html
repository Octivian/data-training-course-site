做 PPT

// from blank to deck in 10 min
Minimax自家 PPT 神器
国内 免费
"一句话需求 → 整套成稿 → 直接演示，国产 PPT 一键生成"
做什么
输入主题或一句话需求，AI 自动生成结构化 PPT + 配图 + 演讲备注，一站式出稿
适合
✓ 工作汇报 / 内部周会 / 临时路演 / 中文场景
不适合
✗ 极端精修设计（需手动二次排版）
怎么用
登录 → 选 "AI 生成 PPT" → 输主题/粘大纲 → 30 秒出稿 → 导出
→ 打开 Minimax
腾讯 ima搜读写一体的 AI
国内 免费
"上传资料 → 总结要点 → 直接生成演示稿"
做什么
搜索 + 读资料 + 写稿 + 生成演示文档四合一，腾讯生态打通
适合
✓ 读研报后做总结稿 / 内部知识沉淀 / 微信生态
不适合
✗ 视觉设计要求高的对外发布
怎么用
小程序或 PC 端 → 丢 5 份资料 → 让它"基于这些写一份 10 页演示"
→ 打开 ima
智谱 Agent多步执行
国内 免费额度
"给它一个目标，自己拆解步骤直到 PPT 出炉"
做什么
Agent 模式：自动拆任务 → 查资料 → 写大纲 → 搜配图 → 拼 PPT，全流程无人值守
适合
✓ 复杂主题（需查资料） / 多步骤任务 / 想"放手让它干"
不适合
✗ 时间卡很死（Agent 跑几分钟到十几分钟）
怎么用
智谱清言 → 切到 "Agent" 模式 → 说"做一份关于 XX 的 12 页 PPT"
→ 打开智谱
腾讯 marvis演示文稿专家
国内 免费
"腾讯出品的 PPT 专用 AI，演示场景深度优化"
做什么
输主题或大纲 → 自动生成演示文稿，对中文字体、版式、行距细节优化好
适合
✓ 中文正式汇报 / 路演 / 模板质量要求高
不适合
✗ 创意设计 / 异形版式
怎么用
marvis.qq.com → 微信登录 → 输主题 → 选风格 → 导出
→ 打开 marvis
豆包 · 专家版字节深度推理
国内 免费
"豆包切到'专家模式'，复杂主题的深度推理 PPT"
做什么
App/网页切到"专家模式"，调豆包大模型 2.0 Pro 做深度推理 → 写大纲 → 出 PPT，逻辑和内容质量上一个档
适合
✓ 行业洞察 / 数据分析 / 长文本报告 / 需要"动脑子"的稿
不适合
✗ 纯视觉设计（用 Minimax / marvis 更合适）
怎么用
打开豆包 → 顶部切"专家模式"→ "给我做一份 XX 主题的 12 页 PPT 大纲 + 关键页内容"
→ 打开豆包
ChatGPT + 提示词万能底座
海外 付费版
"不直接出 PPT，但能 100% 定制结构和文案"
做什么
先让 ChatGPT 输出完整大纲 + 每页文案 + 图表建议，再粘到任意 PPT 工具里
适合
✓ 内容质量要求高 / 逻辑严密 / 行业洞察类
不适合
✗ 时间紧急（需手动排版）
提示词
"我要做一份关于 XX 的汇报，给听众是 XX，希望达到 XX 效果。请按'问题—洞察—方案—收益'四段式输出每页标题和核心要点。"
→ 打开 ChatGPT

ChatGPT综合最强
海外 付费版
"综合能力天花板，中英文都强"
做什么
写所有类型文案：营销文案、邮件、汇报、公关稿、社交媒体

适合
✓ 公众号长文 / 邮件 / 报告 / 中文文风细腻的稿
不适合
✗ 实时信息（无联网）
→ 打开 Claude
Kimi (月之暗面)国内长文
国内 免费
"国产长文之王，能直接读 PDF"
做什么
上传 PDF/Word/网页，AI 帮你读 + 总结 + 写稿。20 万字上下文
适合
✓ 读研报/合同/财报 / 写基于资料的长文
不适合
✗ 复杂推理 / 创意写作
杀手级用法
丢 5 份行业报告进去 → "基于这些资料，写一篇行业分析" → 比 GPT 准
→ 打开 Kimi
豆包 / 通义千问 / 文心一言国产综合

🧭30 秒速查决策树：

Q：我要做一份汇报 PPT，今天就要。
A：先 Gamma 出初稿 → 用 ChatGPT 优化文案 → 套 Slidesgo 咨询风模板 收尾。

Q：我要画一张主视觉海报，要够"高级"。
A：Midjourney 出底图（英文 prompt）→ Canva 加中文文字 → 微调导出。

Q：我要写一篇 2000 字的深度公众号。
A：用 Kimi 读完 5 份参考 → 让 Claude 写初稿（人话最像）→ 自己改 20%。

Q：我要做一份行业调研报告，3 天后交。
A：秘塔 搜全网 → 整理到 Notion/飞书 → 用 ChatGPT 强推理版 梳理框架 → 配艾瑞/QuestMobile 数据。

Q：领导丢我一份 Excel，让我出结论。
A：国内直接用 WPS AI 或 办公小浣熊 丢上去跑分析 + 出图 → 复制结论到 PPT。
